# MLOps Workflow

A simple prototype of my MLOps workflow.

## Tools

- [MLflow](https://mlflow.org/docs/latest/index.html): Experiment tracking and model registery.
- [Optuna](https://optuna.readthedocs.io/en/stable/index.html): Hyperparameter optimization.
- [DVC](https://doc.dvc.org/start): Data versioning and stage building.
- [OmegaConf](https://omegaconf.readthedocs.io/en/2.3_branch/index.html): Parameter management.
- [GX Core (Great Expectation)](https://docs.greatexpectations.io/docs/core/introduction/gx_overview): Data validation.
